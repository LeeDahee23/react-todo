# [드림코딩] 리액트로 투두앱 만들기
💡 <드림코딩 리액트 개념정리 · 클론코딩> 강의 수강 후 실습한 내용을 혼자 다시 만들어보며 복습한 투두앱 입니다. <br/>

### ⚙️ 기능 설명
1. 투두 리스트 보여주기
2. 투두 추가하기
3. 투두 삭제하기
4. 투두 체크하기
5. 투두 리스트 필터링하기
6. 다크모드
7. 로컬 스토리지 저장
